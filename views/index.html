Qu'est-ce que Planifix ?
Planifix est un site web permettant d’enregistrer les films, séries, animes ou événements que l’on compte regarder/assister lorsque la date de sortie arrivera. Profitez de notre base de données riche et constamment mise à jour pour trouver les informations les plus précises sur vos contenus préférés.
